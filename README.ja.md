# GRBL 3Dタッチプローブ用 ストック中心測定マクロ

これは、3Dタッチプローブを使って材料のXY中心を測定するためのGRBL互換プロービングマクロです。

主に、X方向とY方向でそれぞれ反対側を測定することでXY中心を計算できる材料形状を想定しています。  
長方形、正方形、円形の材料などに使用できます。

このマクロでは、まずX方向の反対側を測定し、次にY方向の反対側を測定します。  
その後、計算した中心位置へ移動し、現在のワーク座標系のXY原点を `X0 Y0` に設定します。

プロービングには、`G38.2` や `G10 L20` などのGRBL互換G-codeコマンドを使用しています。

このマクロは、特定の制御ソフト専用ではありません。  
gSender、CNCjs、その他のGRBL制御ソフトでも使用できる可能性がありますが、変数や計算式の書式は各ソフトのマクロ機能に依存します。

## マクロファイル

実際のマクロはこちらです。

[`stock-center-probe-ja.nc`](./stock-center-probe-ja.nc)

## このマクロでできること

- X方向で材料の反対側を測定
- X方向の中心を計算
- X中心へ移動
- Y方向で材料の反対側を測定
- Y方向の中心を計算
- Y中心へ移動
- 現在位置を `X0 Y0` として設定

両側の面を測定して中心を出すため、プローブ球の半径は中心計算上キャンセルされます。

そのため、中心測定においてプローブ径の補正は不要です。

## テスト環境

このマクロは、GRBLベースのデスクトップCNC環境で作成・テストしました。

使用例:

- 機械: LUNYEE 3020 Nova
- コントローラー: GRBLベースのコントローラー
- ファームウェア: GRBL互換 1.1f
- プローブ: 3線式NOタイプの3Dタッチプローブ
- ダイオード: 1N5819ショットキーダイオード推奨、1N4148でも動作します

## 互換性について

このマクロで使用しているプロービングコマンド自体は、GRBL互換の標準的なG-codeコマンドです。

ただし、以下のような変数や計算式の書式は、制御ソフト側のマクロ機能に依存します。

```gcode
%stockX = 200
G0 Z[safeZ]
G38.2 X[search] F[fast]
```

使用している制御ソフトがこの書式に対応していない場合は、そのソフトのマクロ形式に合わせて書き換えるか、変数部分を固定の数値に置き換える必要があります。

## この環境で使用したプローブ配線

![LUNYEE 3020 Novaでの3Dタッチプローブ配線例](./assets/wiring-3d-probe-3020-nova.png)

これは、私の環境でLUNYEE 3020 Novaに3Dタッチプローブを接続するために使用した配線例です。

3Dタッチプローブには以下の3本の線があります。

- Red: 5V
- Black: GND
- Yellow: Probe signal

この配線では、プローブの電源をX-リミットスイッチヘッダーから取り、プローブ信号をダイオード経由でPROBE入力へ接続しています。

1N5819ショットキーダイオードを推奨します。  
1N4148スイッチングダイオードでも動作しますが、順方向電圧が低い1N5819ショットキーダイオードの方が望ましいです。

```text
3D Probe                         3020 Nova

Red   -------------------------> X- header 5V

Black -------------------------> PROBE -

Yellow --- Cathode |<| Anode --> PROBE S
           1N5819 or 1N4148
```

つまり、

- Red: `X- header 5V`
- Black: `PROBE -`
- Yellow: ダイオードのカソード側、帯あり側
- `PROBE S`: ダイオードのアノード側、帯なし側

この配線では、

- 待機中は Yellow 信号線がHighになりますが、PROBE S側へ5Vが直接入りにくくなります
- プローブ反応時は Yellow 信号線がLowになり、ダイオード経由でPROBE SをLowに引っ張ります

これにより、5Vのプローブ信号がPROBE S入力へ直接流れ込むのを防ぎやすくなります。

## 配線上の注意

この配線を使用する前に、必ず自分のコントローラーボードをテスターで確認してください。

この配線では、`PROBE -` と、プローブ電源に使用しているX-ヘッダー側のGNDが共通GNDになっている必要があります。

すべての3020 NovaやGRBLコントローラーボードで、ピン配置や電圧レベルが同じとは限りません。

誤配線により、コントローラーボードやプローブが破損する可能性があります。

この配線は自己責任で使用してください。

また、マクロを実行する前に、制御ソフト上でプローブ入力が正しく検出されていることを確認してください。

プローブ状態が反転している場合は、GRBLの `$6` 設定やプローブ配線を確認してください。

## 実行前の開始位置

![ストック中心測定用プローブ開始位置](./assets/probe-start-position.png)

マクロを実行する前に、以下を確認してください。

1. 材料天面を、GRBL制御ソフトの標準的なZプローブ機能などで `Z0` に設定する
2. プローブを材料の左側外側に配置する
3. Y方向は材料の中央付近に合わせる
4. Zは安全高さにしておく
5. プローブが正しく動作していることを確認する

このマクロは、材料の左側外側・Y中央付近・Z安全高さから開始することを前提にしています。

```text
上面図

                  Y+
                  ^
                  |
                  |

        +-------------------+
        |                   |
   o -> |       材料        |
        |                   |
        +-------------------+

プローブ開始位置:
材料の左側外側、
Y方向は材料の中央付近

X- <---------------------------------> X+
```

## パラメータ

`stock-center-probe-ja.nc` 内の以下の値を、使用する材料や環境に合わせて変更してください。

```gcode
%stockX = 200
%stockY = 100
%probeZ = -2.5
%safeZ = 10.0
%margin = 10.0
%search = 50.0
%fast = 150.0
%slow = 40.0
%retract = 2.0
```

## パラメータ説明

| パラメータ | 説明 |
|---|---|
| `stockX` | 材料のおおよそのX方向寸法、上面図で左右方向 |
| `stockY` | 材料のおおよそのY方向寸法、上面図で手前・奥方向 |
| `probeZ` | 側面測定時のZ高さ |
| `safeZ` | 早送り移動時の安全Z高さ |
| `margin` | 材料外側へ回り込むための余裕量 |
| `search` | プロービング時の最大探索距離 |
| `fast` | 1回目のプロービング送り速度 |
| `slow` | 2回目の低速プロービング送り速度 |
| `retract` | 各測定後の退避距離 |

`stockX` と `stockY` は、主に材料の反対側へ回り込むために使用しています。

最終的な中心計算には直接使われていません。

円形材料の場合、`stockX` と `stockY` はおおよその直径、または材料を囲む外接四角形の寸法として考えると分かりやすいです。

ただし、これらの値が小さすぎると、反対側へ移動したつもりでもプローブがまだ材料上に残ってしまう可能性があります。

その状態でZを下げると、プローブが材料に接触・衝突する可能性があります。

## 中心計算の考え方

このマクロでは、まずX方向で材料の反対側を測定します。

```gcode
%xMinus = params.PRB.x
%xPlus = params.PRB.x
```

その後、X方向の中心を計算します。

```gcode
%centerX = (xMinus + xPlus) / 2
```

X+側測定後の現在位置は、`xPlus + retract` です。

そのため、X中心へ戻るための移動量は以下のようになります。

```gcode
%xMoveToCenter = centerX - (xPlus + retract)
```

Y方向も同じ考え方です。

```gcode
%centerY = (yMinus + yPlus) / 2
%yMoveToCenter = centerY - (yPlus + retract)
```

反対側の2点を測定して中心を出すため、プローブ球の半径は中心計算上キャンセルされます。

## ワーク座標の設定

このマクロでは、以下のコマンドを使用しています。

```gcode
G10 L20 P0 X0 Y0
```

これは、現在使用中のワーク座標系に対して、現在位置が `X0 Y0` になるように設定します。

G54を明示的に設定したい場合は、以下の行を、

```gcode
G10 L20 P0 X0 Y0
```

次のように変更できます。

```gcode
G10 L20 P1 X0 Y0
```

## 安全上の注意

このマクロと配線は自己責任で使用してください。

実際の材料上で実行する前に、以下を必ず確認してください。

- プローブ入力を事前にテストする
- プローブ配線を確認する
- プロービング方向を確認する
- 開始位置が正しいことを確認する
- Zを下げる前に、プローブが材料の外側へ完全に出ていることを確認する
- 初回テスト時は慎重に実行する
- いつでも停止できる状態で実行する
- 材料寸法とマージンに十分な余裕があることを確認する

このマクロはGRBLベースのデスクトップCNC環境で作成・テストしましたが、動作はコントローラー、ファームウェア、制御ソフト、マクロパーサー、プローブ配線、プローブ入力の設定によって変わる可能性があります。
